# Canva 下载接力加速器 (Chrome 扩展程序)

这是一个为 Canva 网页版量身打造的超强免刷新接力下载工具！无需官方复杂的导出操作，自动拦截导出请求并接力完成多页码的高清并发下载。

## 🌟 核心功能

- **🚀 自动拦截与响应**：在原生页面随便点一次下载即可打通隧道。
- **🟢 智能状态跟随**：支持“蓝/黄/绿”三种状态颜色，分别代表选中、排队渲染中、下载完成。
- **💾 永久记忆勾选**：自动保存您的页码选择偏好。
- **🎯 智能吸附与隐藏**：面板跟随 Canva 原生下载窗口智能弹出和收起。
- **🛸 悬浮小球模式**：原生面板关闭时，自动化身为右侧可拖拽小球，即点即出。
- **🕒 5秒优雅退场**：全选任务下载完毕后，悬浮窗显示成果，并在5秒后自动淡出消失。

## 📖 如何安装本插件 (手动加载版)

由于此插件未上架到 Chrome Web Store，请按照以下步骤手动安装：

1. **下载或克隆本仓库**:
   点击 `Code` -> `Download ZIP` 下载源码包，并解压到一个您容易找到的文件夹中。
   _(或者，如果您在 Releases 页面看到了 `CanvaDownloader_vx.x.zip` 包，请下载并解压)_
2. **打开 Chrome 扩展程序页面**:
   在浏览器地址栏输入 `chrome://extensions/` 然后回车。
3. **开启开发者模式**:
   在页面右上角，找到并打开 **“开发者模式”** 开关。
4. **加载已解压的扩展程序**:
   点击左上角的 **“加载已解压的扩展程序”** 按钮。
5. **选择文件夹**:
   选择您刚才解压出来的那个包含 `manifest.json` 的文件夹，点击确定即可完成安装！

## 💡 如何使用

1. 安装好插件后，打开或刷新任意 Canva 设计页面。
2. Canva 界面右上角点击原本的 **“分享” -> “下载”**。
3. 此时插件的紫色**“🚀 Canva 下载加速器”**面板会自动跟着弹出来！
4. 在原生的下载白板中随便**点击一次下载**（此步骤是为了让插件成功劫持您的鉴权 Token 数据）。
5. 看到外挂面板显示 `🟢 隧道就绪，可随时下载` 后，在网格中点击选择你要独立下载的页数（多选）。
6. 最后点击紫色按钮 **“⚡ 开始下载选定的页面”** ！坐和放宽，插件会自动开始排队并完成接力下载作业。
